Adicione imagens e GIFs deste projeto nesta pasta para que sejam exibidos no site.

Sugestões:
- `demo.gif` — GIF mostrando o fluxo principal (ex.: enviar texto -> ver resultado).
- `screenshot1.png`, `screenshot2.png` — screenshots do notebook ou interface.

Depois de adicionar, commit e push:

```bash
git add docs/assets/* && git commit -m "Add demo assets" && git push origin main
```

Se preferir, coloque imagens na raiz `assets/` e atualize `docs/index.html` conforme necessário.
