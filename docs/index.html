Análise de Sentimentos

Projeto demonstrativo para classificar sentimento de textos (positivo / neutro / negativo). Ideal para portfólio.

Funcionalidades

  • Classificação de sentimento
  • Pipeline de pré-processamento (limpeza e tokenização)
  • Modelos treinados e script de inferência
  • API simples para inferência (opcional)

Como publicar com GitHub Pages

  1. Confirme que os arquivos estão no branch `main`.
  2. Coloque esta página em `docs/index.html` (já feita) e os assets em `docs/assets/`.
  3. Commit e push:
git add docs/ && git commit -m "Add GitHub Pages site" && git push origin main

Depois, no GitHub: Settings → Pages → Source → Branch: main / folder: /docs. Salve — o site ficará disponível em https://<seu-user>.github.io/<repo> em alguns minutos.

Como rodar localmente

Este repositório contém principalmente o material do projeto. Para rodar uma API ou notebook local, adicione seus arquivos em `src/` ou `notebooks/` e siga as instruções do `README.md`.
